# Software Engineering Portfolio

A minimalist, high-performance professional portfolio designed to showcase core software development competencies, data engineering architectures, and system documentation standards.

---

## Technical Stack
* **Frontend Framework:** Vue.js 3 (Composition API)
* **Build Tool:** Vite
* **Styling & Design:** Tailwind CSS v4
* **Deployment:** Vercel

---

## Core Projects Highlighted
* **Cosmic CatStudio:** A modular client-server platform built with PHP and MySQL, featuring full documentation and system architecture blueprints.
* **Data Engineering Pipelines:** ETL orchestration architectures utilizing Apache Hop and containerized environments with Docker Compose.
* **Computer Vision Systems:** Automated object detection algorithms using YOLOv8 and advanced morphological operations with OpenCV in Python.

---

## Architecture & Project Structure
* `/src/views`: Contains the primary single-page modules (Hero, About, Projects, Contact).
* `/src/components`: Reusable UI elements and global layout controllers like the Navigation Bar.
* `/src/assets`: Production static assets, optimized tech stack iconography, and project previews.
* `/public`: Direct distribution directory housing static enterprise assets, including the professional CV.
